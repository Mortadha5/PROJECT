================================================================================
1. RÉSUMÉ COMPLET DU PROJET
================================================================================

Ce projet est une plateforme web intelligente de recommandation de formations
professionnelles basée sur le Machine Learning. Elle permet de prédire la
formation la plus adaptée à un employé en fonction de son âge, son expérience
et ses compétences techniques.

L'application combine trois domaines clés :
- Intelligence Artificielle (modèle de prédiction Random Forest)
- Cybersécurité (authentification biométrique WebAuthn, gestion des sessions,
  audit de sécurité, blocage d'utilisateurs)
- DevOps (conteneurisation Docker, architecture multi-conteneurs,
  déploiement automatisé)

Le projet est structuré en architecture client-serveur avec :
- Un backend Flask servant d'API REST
- Un frontend React (Vite) avec interface moderne
- Une base de données MongoDB
- Un système de notifications en temps réel via WebSocket (Socket.IO)


================================================================================
2. OBJECTIF PRINCIPAL DE L'APPLICATION
================================================================================

Fournir aux entreprises un outil intelligent de gestion des compétences et
de recommandation de formations, permettant :

- Aux employés : de soumettre leur profil et recevoir des recommandations
  de formations personnalisées avec un roadmap d'apprentissage.
- Aux administrateurs : de gérer les utilisateurs, valider les prédictions,
  suivre l'historique, monitorer la sécurité, et communiquer avec les
  utilisateurs.

L'objectif est d'optimiser le processus de montée en compétences au sein
d'une organisation en utilisant l'IA pour personnaliser les parcours de
formation.

3. FONCTIONNALITÉS DÉJÀ DÉVELOPPÉES (TERMINÉES)
================================================================================

3.1 AUTHENTIFICATION & GESTION D'ACCÈS
- Connexion/déconnexion avec session sécurisée
- Gestion des rôles (admin/user) avec décorateurs @login_required, @admin_required
- Authentification biométrique WebAuthn (empreinte digitale / Windows Hello)
- Réinitialisation de mot de passe (par l'admin)
- Changement de mot de passe par l'utilisateur
- Blocage/déblocage d'utilisateurs
- Journalisation de toutes les actions (audit trail)

3.2 PRÉDICTION & MACHINE LEARNING
- Modèle Random Forest Classifier (200 estimateurs)
- Encodage one-hot des compétences multiples
- Prédiction simple (une compétence → une formation)
- Prédiction avancée (multi-compétences → top 5 formations avec scores)
- Analyse de profil utilisateur (stade carrière, capacité d'apprentissage)
- Skill Gap Analysis (compétences acquises, manquantes, recommandées)
- Score de confiance personnalisé par recommandation
- Raisons explicatives pour chaque recommandation
- Workflow de validation : user soumet → admin review → envoi du résultat

3.3 CHATBOT IA
- Chatbot basé sur des règles (keyword matching)
- Réponses sur les formations, salaires, durées, conseils
- Personnalisation des réponses avec le nom de l'utilisateur
- Système de feedback (helpful / not_helpful)
- Sauvegarde des conversations en base
- Statistiques d'utilisation du chatbot (admin)

3.4 MESSAGERIE EN TEMPS RÉEL
- Messagerie bidirectionnelle user ↔ admin
- Notifications en temps réel via Socket.IO
- Compteur de messages non lus (badge)
- Historique des conversations
- Marquage de lecture automatique

3.5 NOTIFICATIONS
- Système de notifications push en temps réel (Socket.IO)
- Notifications individuelles et broadcast (à tous)
- Types : info, success, prediction, admin_message, broadcast
- Compteur de notifications non lues (badge)
- Marquer comme lu / Supprimer
- Envoi automatique lors d'événements (nouvelle prédiction, message, etc.)

3.6 ROADMAP DE FORMATION
- Roadmaps personnalisées pour 10+ formations
- Phases d'apprentissage structurées (4 phases par formation)
- Suivi de progression par compétence (checkbox par skill)
- Sauvegarde de la progression en base MongoDB
- Conseils personnalisés selon l'âge et l'expérience
- Ressources et certifications recommandées
- Administration des roadmaps (vue globale, suppression)

3.7 ADMINISTRATION
- Dashboard avec statistiques (nb utilisateurs, prédictions, âge moyen, etc.)
- Distribution des formations (graphiques)
- Gestion des utilisateurs (CRUD complet)
- Historique de toutes les prédictions avec filtres
- Interface de sécurité (logs, activités suspectes, blocage)
- Gestion des prédictions en attente (review workflow)
- Gestion des roadmaps utilisateurs

3.8 PROFIL UTILISATEUR
- Consultation et édition du profil
- Upload d'avatar (photo ou icône aléatoire)
- Statistiques personnelles (nb prédictions, messages chatbot)
- Activité récente
- Durée de membership

3.9 FRONTEND REACT MODERNE
- Single Page Application (React + Vite)
- Interface moderne avec design glassmorphism
- Sidebar collapsible avec hover-to-reveal
- Responsive design (desktop + mobile)
- Badges de notification en temps réel
- Routage protégé par rôle (ProtectedRoute)
- Toasts de notification (react-hot-toast)
- Thème sombre cohérent (purple/blue gradient)

12. MODÈLES MACHINE LEARNING UTILISÉS
================================================================================

MODÈLE PRINCIPAL : Random Forest Classifier (scikit-learn)
─────────────────────────────────────────────────────────────────────────────
- Algorithme         : RandomForestClassifier
- Nombre d'arbres    : 200 (n_estimators=200)
- Profondeur max     : 12 (max_depth=12)
- min_samples_split  : 5
- min_samples_leaf   : 2
- Pondération classes: balanced (class_weight='balanced')
- Random state       : 42

FEATURES D'ENTRÉE :
- age (numérique)
- experience (numérique)
- Vecteur one-hot des compétences (N dimensions, N = nombre de compétences)

SORTIE :
- Formation recommandée (classification multi-classes, 10+ classes)
- Probabilités pour chaque formation (predict_proba)

DATASET :
- Fichier : dataset_employes_formation_complet.csv
- Colonnes : age, experience, competence (multi-valeurs séparées par virgule),
             formation_suggeree

FICHIERS MODÈLE SÉRIALISÉS :
- model.pkl          : Modèle entraîné
- comp_dict.pkl      : Dictionnaire compétences → index
- form_dict.pkl      : Dictionnaire formations → encodage
- inv_form_dict.pkl  : Dictionnaire inverse (index → formation)
- all_competences.pkl: Liste de toutes les compétences

MÉTRIQUES :
- Validation croisée (5-fold) effectuée
- Classification report généré à l'entraînement

POST-TRAITEMENT PERSONNALISÉ :
- Score personnalisé combinant : probabilité ML + correspondance compétences
  + ajustements âge/expérience + opportunités marché
- Analyse de profil (stade carrière, capacité apprentissage)
- Skill Gap Analysis (compétences acquises vs manquantes)

15. PARTIES LIÉES À LA CYBERSÉCURITÉ
================================================================================

15.1 AUTHENTIFICATION
- Hachage bcrypt avec salt automatique (pas de stockage en clair)
- Authentification biométrique WebAuthn (FIDO2) : empreinte digitale,
  Windows Hello, clé de sécurité USB
- Sessions sécurisées : HTTPOnly, SameSite=Lax, durée limitée (2h)
- Système de blocage de comptes

15.2 CONTRÔLE D'ACCÈS (RBAC)
- Deux rôles : admin et user
- Décorateurs @login_required et @admin_required
- Routes protégées côté backend ET frontend (ProtectedRoute)
- Empêchement de suppression du compte admin

15.3 AUDIT & MONITORING
- Journalisation complète : login, logout, login_failed, login_blocked,
  user_create, user_delete, password_change, biometric_setup, etc.
- Détection d'activités suspectes (3+ tentatives échouées)
- Tableau de bord sécurité avec logs temps réel
- Nettoyage programmable des anciens logs

15.4 VALIDATION & PROTECTION
- Validation des entrées (longueur username, format email, etc.)
- Secure filename pour uploads (werkzeug.utils.secure_filename)
- Limitation taille uploads (16 MB max)
- Extensions fichiers autorisées (whitelist : png, jpg, jpeg, gif)
- CORS configuré (origins limitées)

15.5 POINTS D'ATTENTION
- SECRET_KEY hardcodée par défaut (à changer en production)
- SESSION_COOKIE_SECURE=False (nécessite HTTPS en production)
- Protection CSRF commentée (non active)
- Debug prints présents (à retirer en production)

16.2 SYSTÈME DE RECOMMANDATION AVANCÉ
- Probabilités pour toutes les classes (predict_proba)
- Score personnalisé combinant :
  * Probabilité ML de base
  * Correspondance compétences requises (+15 max)
  * Correspondance compétences liées (+5 max)
  * Ajustements âge et expérience (+5 à +8)
  * Bonus opportunités marché (+5 à +10)
- Top 5 recommandations classées par score

16.3 ANALYSE INTELLIGENTE DU PROFIL
- Classification par stade de carrière (débutant → senior)
- Évaluation de la capacité d'apprentissage
- Estimation de la disponibilité temporelle
- Évaluation de la tolérance au risque
- Force du profil (polyvalent, diversifié, spécialisé)

16.4 SKILL GAP ANALYSIS
- Graphe de compétences (COMPETENCE_GRAPH) avec relations
- Identification compétences acquises vs manquantes
- Score de couverture (%)
- Recommandations de compétences à acquérir

16.5 CHATBOT (NLP basique)
- Reconnaissance d'intentions par mots-clés
- Matching de formations dans le message
- Réponses contextuelles et personnalisées
- Note : pas de modèle de langage (LLM), règles statiques

16.6 ROADMAPS PERSONNALISÉES
- Adaptation de la roadmap selon le profil
- Rythme d'apprentissage adapté (progressif/normal/accéléré)
- Conseils spécifiques par tranche d'âge et expérience
